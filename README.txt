RugFlag — Favicons Pack (v1)

Files:
- favicon.ico              → put in the ROOT of /RugFlagFixed (same level as index.html)
- apple-touch-icon.png     → also put in ROOT (iOS often picks it automatically)
- favicon-512.png          → optional, future PWA/manifest usage

Installation (no HTML edits):
1) Upload the three files above to the repository root that GitHub Pages serves.
2) Open https://merxio-nl.github.io/RugFlagFixed/favicon.ico?ver=1 to bust cache.
3) Reload the site on mobile/desktop.

Optional hard links (if ever needed):
<link rel="icon" href="favicon.ico">
<link rel="apple-touch-icon" href="apple-touch-icon.png">
